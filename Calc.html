* 자동견적기 사용방법

• 자동견적기에 사이즈 입력 후 나온 구매수량을 입력하면 주문 끝

ex) 120 x 140 = 37개 (옵션선택창에 37개로 수량을 맞춰주세요)

• 구매수량이 여러개일 경우 각 수량을 총 합하여 주문

ex) 120 x 140 2장 필요시

구매수량 37개 x 2장 = 74개 주문

(배송메모란에 정확한 사이즈 필수 입력. 미입력시 1장으로 합산되어 제작)
